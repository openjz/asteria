1. 容器类型

  • 顺序容器
    • 连续内存
      • vector
      • deque：双端队列
      • array：固定大小
      • string
    • 链表
      • list：双向链表
      • forward_list：单向链表

2. 和容器大小有关的操作

  • 大小
    • empty
    • size
    • max_size：返回该类型容器能容纳的最大元素数量
    • resize：改变容器大小
  • 容量
    • capacity：返回容器的容量，即不重新分配内存的情况下，容器最多能保存的元素
    • reserve：不改变容器大小，改变容器的容量（只能扩大，不能缩小）
    • shrink_to_fit：将容量缩小到和大小一致

3. 内存操作

  • assign：内存拷贝
  • swap：内存交换
